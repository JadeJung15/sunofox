    사용방법

    SF Studio는 이미 만든 30컷 프롬프트와 후보 이미지를 정리하는 작업실입니다. 생성 순서는 편하게 관리하고, 최종 영상 순서는 편집 단계에서 음악과 가사에 맞춰 직접 결정합니다.

    Workflow MD 가져오기
    문제 해결 현재 증상과 같은 항목을 열고 확인 순서대로 점검하세요.
    Workflow 가져오기 실패컷을 찾지 못하거나 검증 오류가 계속 표시될 때
    증상
    가져오기 버튼을 눌러도 콘티가 열리지 않거나 컷 번호, time, MJ, Grok 누락 메시지가 표시됩니다.
    확인
    ## Cut 01 제목과 Workflow MD 원문 전체가 포함됐는지, 가져오기 미리보기의 오류 목록을 먼저 확인합니다.
    해결
    원본에서 누락 필드를 보완한 뒤 전체 내용을 다시 붙여넣습니다. 기존 MJ Markdown은 코드 블록이 포함된 Cut 원문을 사용하세요.
    Workflow MD 가져오기 열기
    SF 미디어 브릿지 연결 또는 이미지 수집 문제전송 응답이 없거나 후보 이미지가 들어오지 않을 때
    증상
    Midjourney 전송이 시작되지 않거나 생성된 이미지가 후보 이미지 보관함에 수집되지 않습니다.
    확인
    같은 Chrome에 SF 미디어 브릿지 1.5.23이 설치·활성화되어 있는지, 콘티 제작의 브릿지 상태와 대상 탭을 확인합니다.
    해결
    브릿지 상태 확인을 다시 실행하고 대상 Midjourney 또는 Grok 탭을 연 뒤 재시도합니다. 버전이 다르면 아래 ZIP으로 교체하세요.
    브릿지 1.5.23 ZIP 다운로드
    저장 프로젝트 복구 또는 초기화저장 작업이 보이지 않거나 깨끗하게 다시 시작할 때
    증상
    SF 홈에 최근 작업이 표시되지 않거나 불러온 뒤 첨부 이미지 일부가 비어 있습니다.
    확인
    같은 브라우저와 Google 오너 계정인지 확인하고, 콘티 제작의 저장 상태와 백업 JSON 보유 여부를 확인합니다.
    해결
    저장 작업 불러오기 또는 백업 불러오기를 먼저 사용합니다. 완전히 다시 시작할 때만 초기화를 누르고, 중요한 작업은 먼저 이미지 포함 백업으로 내보내세요.
    내보내기 또는 파일 형식 문제백업, Markdown, CSV 파일이 열리지 않을 때
    증상
    내보낸 파일에 원하는 데이터가 없거나 백업 불러오기에서 파일 형식 오류가 표시됩니다.
    확인
    작업 복구용은 이미지 포함 JSON 백업, 프롬프트 공유용은 Markdown, 컷리스트용은 CSV인지 목적과 확장자를 확인합니다.
    해결
    복구에는 SF Studio가 내보낸 JSON 원본을 수정하지 않고 사용합니다. Excel에서 CSV가 깨지면 UTF-8로 가져오고, 프롬프트 전달은 Markdown을 사용하세요.
    기본 작업 순서처음 사용할 때의 빠른 기준
    1. MV 컷 제작 도구에서 Workflow MD 또는 기존 MJ Markdown을 만듭니다.
    2. Workflow MD 화면에서 원문을 가져오고 검증 메시지를 확인합니다.
    3. 콘티 제작에서 Midjourney로 보내고 후보 이미지를 모읍니다.
    4. 이미지 선택 후 Grok 클립을 만들고 최종 순서는 편집 단계에서 확정합니다.
    보존 기준결과물이 꼬이지 않게 유지합니다.
    • 원문 보존Workflow MD의 MJ/Grok 프롬프트는 자동 재작성하지 않습니다. 기존 MJ Markdown은 전송 옵션만 정리합니다.
    • 후보 관리30컷을 넘어도 후보 컷을 추가할 수 있으며 최종 영상 순서는 음악과 가사 기준으로 편집 단계에서 정합니다.
    • Grok Saved비디오 옵션을 먼저 맞춘 뒤 이미지를 첨부합니다. 고급 실행에서는 처음부터, 현재부터, 구간 제작과 대기 시간을 조정할 수 있습니다.
    화면별 역할필요한 화면과 자동화 기준
    • SF 홈외부 제작 도구와 최근 저장 작업으로 이동합니다.
    • Workflow MD원문을 붙여넣고 컷별 MJ, Grok, 편집 메모를 분리합니다.
    • 콘티 제작컷별 전송, 후보 이미지 첨부, Grok 클립 준비를 처리합니다.
    • Grok Saved 매크로브릿지가 이미지와 프롬프트를 채우고 Enter를 누릅니다. 제출 후 Saved 화면 복귀가 확인되면 다음 컷을 시작합니다.
    • 앱 설치SF Studio는 앱처럼 설치할 수 있으며 자동화에는 같은 브라우저의 SF 미디어 브릿지가 필요합니다.

    고화질 MV 제작

    기본 사용 순서 처음 사용할 때만 확인
    1. MV 컷 제작 도구에서 만든 Workflow MD를 가져옵니다.
    2. 컷별 MJ/Grok 프롬프트와 편집 메모를 확인합니다.
    3. Midjourney로 보내기 또는 일괄 추출을 사용합니다.
    4. 선택한 이미지를 후보 이미지에 모아 Grok 클립 제작으로 넘깁니다.
    브릿지 설치 파일 다른 PC에서 사용할 때만 열기

    ZIP 파일을 내려받아 압축을 푼 뒤, Chrome 확장 프로그램에서 압축 해제된 폴더를 로드합니다.

    브릿지 ZIP 다운로드
    1. 다운로드한 ZIP 압축을 풉니다.
    2. Chrome 주소창에 chrome://extensions를 입력합니다.
    3. 개발자 모드를 켠 뒤, 압축 해제된 midjourney-bridge 폴더를 로드합니다.
